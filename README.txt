========================================
  视频下载工具 - 使用说明
========================================

📦 文件说明：
  main.py         后端服务程序
  static/         前端页面
  downloads/      下载文件保存位置
  START.bat       启动服务（双击运行）

▶ 如何启动：
  1. 双击 START.bat
  2. 看到 "Uvicorn running on http://0.0.0.0:16888" 即启动成功
  3. 浏览器打开 http://localhost:16888

▶ 使用：
  粘贴视频链接 → 点击"获取信息" → 选择画质 → 点击"开始下载"

▶ 支持平台：
  YouTube       ✅ 直接下载，无需任何配置
  B站 (Bilibili)  ✅ 直接下载
  抖音          ⚠️ 部分需要登录
  TikTok        ⚠️ 部分需要登录
  Instagram     ⚠️ 部分需要登录
  快手          ✅ 直接下载
  微博          ⚠️ 部分需要登录
  视频号        ⚠️ 需要登录
  小红书        ⚠️ 部分需要登录

▶ 关闭服务：
  按 Ctrl+C 或者直接关掉命令行窗口

▶ 需要 yt-dlp：
  如果提示"未安装 yt-dlp"，在命令行执行：
  pip install yt-dlp
